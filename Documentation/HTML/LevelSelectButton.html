public class LevelSelectButton


Namespace: Starborne.Control

Inherits from: MonoBehaviour


Class that manages the level select buttons. These buttons can be pushed to load the given level.



private stringscenePathPath to the scene's JSON-file.
private intstarsAmount of stars achieved in this level.
private GameObject[]starsImagesArray of the GameObjects with the star images.
private SpritestarSpriteThe Sprite that will be displayed if the player has gotten a star.
private SpritenoStarSpriteThe Sprite that will be displayed if the player has not gotten a star.
private boolisUnlockedWhether or not the player has unlocked this level.


public voidSetScenePath(string newScenePath)Assigns new value to scenePath.
public voidSetStarCount(int starCount)Assigns new value to stars.
public voidSetUnlocked(bool state)Assigns new value to isUnlocked.
public stringGetScenePath()Returns scenePath
public voidAttemptSubmit()Attempt to enter the given level. If isUnlocked is true, call SendPathToSceneDataHandler.
private voidSendPathToSceneDataHandler()Find SceneDataHandler and save the scenePath with SetSceneData. Also, load the loading-scene.
