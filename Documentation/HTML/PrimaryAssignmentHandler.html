public class PrimaryAssignmentHandler


Namespace: Starborne.Mission

Inherits from: MonoBehaviour

Implements interfaces: ILateInit


Class that handles the primary assignment of the player. The primary assignment is the same in every level: destroy all the enemies.



private floatchangeSceneOnWinDelayAmount of seconds between winning the game and loading the next scene.
private intenemyCountAmount of enemies when the game starts.
private intenemiesKilledAmout of enemies killed.
private EnemyHealth[]enemiesArray of the enemies.
private GameUIgameUIThe GameUI.


public voidLateAwake()Find values for gameUI, enemies, enemyCount, and set enemiesKilled to 0.
public voidLateStart()Add EnemyDestroyed to every enemy's onDeath event. Also, call CheckWin.
private voidEnemyDestroyed()Is called when an enemy's onDeath is invoked. Increments enemiesKilled and calls CheckWin.
private voidCheckWin()Checks if the player has won and updates the UI. If enemiesKilled is equal to enemyCount, call Win.
private voidUpdateUI()Update the mission text.
private voidWin()This function is called when the player wins the game. Finds the OptionalAssignmentHandler and calls CaptureData and SetLevelWon(true), unlocks the cursor, finds the SceneHandler, and loads the game-over scene.
